## 系统概述

AI创作者平台是一个集成多种AI创作工具的综合平台，提供AI写作、图片生成、视频生成、PPT生成等功能，并支持一键发布到多个平台。

### 技术栈
- **后端**: FastAPI + Python 3.10+ + SQLAlchemy + MySQL 8.0+
- **前端**: Vue 3 + TypeScript + Vite + Element Plus
- **AI服务**: OpenAI、Anthropic、百度文心、智谱AI、阿里通义千问
- **缓存**: Redis
- **异步任务**: Celery

### 核心特性
- ✅ 14种专业AI写作工具
- ✅ AI图片生成和编辑
- ✅ AI视频生成
- ✅ AI PPT生成
- ✅ 一键多平台发布
- ✅ 积分和会员系统
- ✅ 运营管理系统（活动、优惠券、推广返利）
- ✅ 完整的用户权限管理

## 核心功能

### 1. AI写作工具（14个专业工具）

#### 1.1 公众号文章创作
- **功能**: 根据主题生成适合微信公众号的文章
- **特点**: 自动优化排版、添加表情符号、SEO优化
- **输入**: 文章主题、关键词、目标读者
- **输出**: 完整的公众号文章（标题+正文+摘要）

#### 1.2 小红书笔记创作
- **功能**: 生成小红书风格的种草笔记
- **特点**: 表情符号丰富、标签优化、图文结合建议
- **输入**: 产品/主题、卖点、目标人群
- **输出**: 小红书笔记（标题+正文+标签）

#### 1.3 公文写作
- **功能**: 生成规范的公文文档
- **特点**: 符合公文格式规范、用语严谨
- **输入**: 公文类型、主题、要点
- **输出**: 规范的公文文档

#### 1.4 论文写作
- **功能**: 辅助学术论文写作
- **特点**: 学术规范、引用格式、逻辑严密
- **输入**: 论文主题、研究方向、关键点
- **输出**: 论文大纲或章节内容

#### 1.5 营销文案
- **功能**: 生成营销推广文案
- **特点**: 吸引眼球、转化导向、多场景适配
- **输入**: 产品信息、卖点、目标客户
- **输出**: 营销文案（标题+正文+CTA）

#### 1.6 新闻稿/软文
- **功能**: 生成新闻稿或软文
- **特点**: 新闻性、可读性、传播性
- **输入**: 新闻主题、关键信息、发布渠道
- **输出**: 新闻稿或软文

#### 1.7 短视频脚本
- **功能**: 生成短视频拍摄脚本
- **特点**: 分镜头、台词、场景描述
- **输入**: 视频主题、时长、风格
- **输出**: 完整的视频脚本

#### 1.8 故事/小说创作
- **功能**: 创作故事或小说内容
- **特点**: 情节连贯、人物鲜明、语言生动
- **输入**: 故事类型、主题、人物设定
- **输出**: 故事或小说章节

#### 1.9 商业计划书
- **功能**: 生成商业计划书
- **特点**: 结构完整、数据支撑、逻辑清晰
- **输入**: 项目信息、市场分析、财务预测
- **输出**: 商业计划书各章节

#### 1.10 工作报告
- **功能**: 生成工作总结或报告
- **特点**: 条理清晰、数据准确、重点突出
- **输入**: 工作内容、成果、问题
- **输出**: 工作报告

#### 1.11 简历/求职信
- **功能**: 生成个人简历或求职信
- **特点**: 专业规范、突出优势、针对性强
- **输入**: 个人信息、工作经历、求职意向
- **输出**: 简历或求职信

#### 1.12 教案/课件
- **功能**: 生成教学教案或课件内容
- **特点**: 教学目标明确、内容系统、易于理解
- **输入**: 课程主题、教学目标、学生水平
- **输出**: 教案或课件内容

#### 1.13 内容改写/扩写/缩写
- **功能**: 对现有内容进行改写、扩写或缩写
- **特点**: 保持原意、风格可调、质量提升
- **输入**: 原文、改写要求、目标长度
- **输出**: 改写后的内容

#### 1.14 多语言翻译
- **功能**: 多语言翻译和本地化
- **特点**: 准确流畅、保持语境、文化适配
- **输入**: 原文、目标语言、专业领域
- **输出**: 翻译后的内容

## AI模型集成

### 支持的AI服务商

#### 1. OpenAI
- **模型**: GPT-4, GPT-3.5-turbo, DALL-E 3
- **用途**: 文本生成、图片生成
- **配置**: API Key

#### 2. Anthropic (Claude)
- **模型**: Claude 3 Opus, Claude 3 Sonnet
- **用途**: 文本生成、长文本处理
- **配置**: API Key

#### 3. 百度文心一言
- **模型**: ERNIE-Bot 4.0, ERNIE-Bot-turbo
- **用途**: 中文文本生成
- **配置**: API Key + Secret Key

#### 4. 智谱AI (ChatGLM)
- **模型**: GLM-4, GLM-3-turbo
- **用途**: 中文文本生成
- **配置**: API Key

#### 5. 阿里通义千问 (Qwen) ✨新增
- **模型**: qwen-max, qwen-plus, qwen-turbo
- **用途**: 中文文本生成、多模态
- **配置**: API Key (DashScope)
- **特点**: 
  - 支持长文本（最高32K tokens）
  - 中文理解能力强
  - 性价比高
  - 支持流式输出

### 模型管理功能

#### 模型配置
- 添加/编辑/删除AI模型
- 配置API密钥（加密存储）
- 设置模型参数（temperature、max_tokens等）
- 启用/禁用模型

#### 模型选择
- 用户可选择使用的模型
- 管理员可设置默认模型
- 支持模型降级和故障转移

#### 使用监控
- API调用次数统计
- Token使用量统计
- 成本分析
- 错误率监控

## 积分会员系统

### 1. 积分系统

#### 1.1 积分充值
- **充值方式**: 在线支付（支付宝、微信）
- **充值比例**: 1元 = 10积分
- **最低充值**: 10元（100积分）
- **支付回调**: 自动到账

#### 1.2 积分消费
- **消费规则**: 
  - 非会员用户每次生成消耗10积分
  - 会员用户不消耗积分
- **消费记录**: 详细的消费明细
- **余额查询**: 实时积分余额

#### 1.3 积分退款
- **退款场景**: 生成失败自动退款
- **退款方式**: 原路退回积分
- **退款记录**: 完整的退款日志

#### 1.4 积分赠送
- **注册赠送**: 新用户注册赠送积分
- **活动赠送**: 参与活动获得积分
- **推荐奖励**: 推荐新用户获得积分

### 2. 会员系统

- ✅ 专属客服支持

#### 2.3 会员管理

## 运营管理系统

- 目标用户群体

#### 1.4 活动参与
- 用户参与记录
- 奖励发放
- 参与次数限制
- 参与统计

### 2. 优惠券系统

#### 2.1 优惠券类型
- **充值优惠券**: 充值时抵扣金额

#### 2.2 优惠券配置
- 优惠券名称和描述
- 优惠券类型

#### 2.3 优惠券状态

#### 2.4 优惠券发放

#### 3.4 返利发放

## 业务流程说明

### 1. 用户注册流程
1. 用户填写注册信息（用户名、邮箱、密码）
2. 系统验证信息有效性
3. 创建用户账号
4. 生成唯一推荐码
5. 检查是否使用推荐码注册
6. 发放新用户注册奖励（积分）
7. 如有推荐人，记录推荐关系
8. 返回注册成功和JWT Token

### 2. 内容生成流程（非会员）
1. 用户选择创作工具并输入内容
2. 系统检查用户会员状态
3. 非会员用户检查积分余额（需要10积分）
4. 积分不足提示充值
5. 积分充足则扣除10积分
6. 调用AI模型生成内容
7. 生成成功返回内容
8. 生成失败自动退还积分
9. 记录交易和创作历史

### 3. 内容生成流程（会员）
1. 用户选择创作工具并输入内容
2. 系统检查用户会员状态
3. 会员用户直接调用AI模型（不扣积分）
4. 生成内容并返回
5. 记录创作历史（不记录积分交易）

### 4. 积分充值流程
1. 用户选择充值金额
2. 系统创建充值订单
3. 生成支付二维码/链接
4. 用户完成支付
5. 支付平台回调通知
6. 系统验证支付结果
7. 充值成功增加积分
8. 记录充值订单和交易
9. 通知用户充值成功

### 5. 会员购买流程
1. 用户选择会员套餐
2. 检查是否有可用优惠券
3. 系统创建会员订单
4. 生成支付二维码/链接
5. 用户完成支付
6. 支付平台回调通知
7. 系统验证支付结果
8. 开通会员权限
9. 记录会员订单
10. 通知用户开通成功

### 6. 活动参与流程
1. 用户查看活动列表
2. 选择感兴趣的活动
3. 检查参与条件和限制
4. 提交参与申请
5. 系统验证参与资格
6. 记录参与信息
7. 发放活动奖励
8. 通知用户奖励到账

### 7. 优惠券使用流程
1. 用户在充值/购买会员时选择优惠券
2. 系统验证优惠券有效性
3. 检查使用条件（门槛金额）
4. 计算优惠后金额
5. 创建订单（优惠后价格）
6. 用户完成支付
7. 标记优惠券为已使用
8. 记录使用记录

### 8. 推广返利流程
1. 用户生成/获取推荐码
2. 分享推荐码给好友
3. 好友使用推荐码注册
4. 系统记录推荐关系
5. 好友完成首次充值/购买会员
6. 触发返利规则
7. 计算返利金额
8. 发放返利奖励
9. 通知推荐人和被推荐人

### 9. 多平台发布流程
1. 用户选择要发布的内容
2. 选择目标发布平台
3. 系统自动适配内容格式
4. 检查平台规则和限制
5. 预览发布效果
6. 确认发布
7. 调用平台API发布
8. 记录发布状态
9. 通知用户发布结果
