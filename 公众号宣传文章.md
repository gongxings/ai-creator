## 🛠️ 技术牛在哪？开发者必看

虽然功能强大，但代码结构**极其优雅**，堪称教科书级别：

### 📂 **标准分层架构**
```
backend/
├── api/v1/          # 路由层（仅验证参数）
├── models/          # 数据层（SQLAlchemy）
├── schemas/         # 验证层（Pydantic v2）
├── services/        # 业务层（核心逻辑）
└── core/            # 基础设施
```

### 🎨 **设计模式应用**
- **工厂模式**：AIServiceFactory动态创建服务  
- **策略模式**：各AI服务实现统一接口  
- **双轨模式**：API Key + Cookie两种调用方式  

### 🔐 **安全规范**
- bcrypt密码加密
- AES API密钥加密存储
- JWT Token + 刷新机制
- SQL注入防护（SQLAlchemy）
- 支付回调签名验证

### 🐳 **一键部署**
```bash
git clone <repo>
docker-compose up -d
# 访问 http://localhost:5173
```

**前后端分离** + **Docker容器化** + **Nginx反向代理**，部署一条龙！
